Getting Started

  1. Upload your TML file
  2. Select the type of update you want to make
  3. Make your changes in the provided interface
  4. Preview your changes
  5. Download the updated TML file

Frequently Asked Questions

What file types does TML Updater support?

TML Updater supports .tml files used by ThoughtSpot.

Can I undo changes made to my TML file?

Yes, you can revert changes before downloading the updated file.

Best Practices

  • Always backup your original TML files before making changes
  • Use descriptive names for your updated files
  • Test your updated TML files in a non-production environment first

Glossary

TML
ThoughtSpot Modeling Language, used to define data models in ThoughtSpot
Join
A connection between two or more tables in a data model

Troubleshooting

If you encounter any issues while using TML Updater, try the following:

  • Ensure your TML file is correctly formatted
  • Clear your browser cache and try again
  • Check our FAQ section for common issues and solutions
